# ARCHON Directives

> This file acts exactly like `.gemini.md` in the Gemini CLI. The Manager Ordinance will permanently attach these guidelines to all prompts.

## Architecture Rules

- Use only vanilla HTML/CSS along with vanilla JavaScript where possible to reduce dependencies.
- No TailwindCSS without explicit request.
- Keep agent functions strictly decoupled from Manager classes.

## Output Tone

- Respond in a concise, authoritative tone. Do not use conversational filler.
